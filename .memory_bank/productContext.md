# Contexto del Producto

## Propósito
Sistema de inventario moderno para Comintec, migrado de una aplicación PHP legacy a una arquitectura React + PocketBase.

## Problemas que Resuelve
- Gestión ineficiente de inventario.
- Interfaz de usuario anticuada y difícil de usar.
- Dificultad para mantener y extender el código legacy.
- Falta de pruebas automatizadas.

## Funcionalidades Clave
- Gestión de Productos (CRUD).
- Gestión de Categorías.
- Gestión de Ventas y Reportes.
- Gestión de Usuarios y Roles.
- Autenticación segura.
- Dashboard con métricas clave.

## Stack Tecnológico
- **Frontend**: React, Vite, TailwindCSS, Emotion.
- **Backend**: PocketBase (Go + SQLite).
- **Infraestructura**: Docker, Docker Compose.
- **Testing**: Vitest (Frontend), Go tests (Backend - si aplica).
